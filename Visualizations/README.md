The **Visualizations** folder contains graphs and presentations resulting from the analysis phase and can be divided into four sub-types:
* _Election_Comparison_Bar_Graph_ images show percentage comparisons of the Standard Winner-Take-All, Nebraska-Like, and Full Divide methods using both Capped and Uncapped Electoral College totals, to the Popular Vote percentage of their election year

* _Top_Methods_and_Candidates_ images display graphs with the comparing the percentages of Nebraska-Like and Full Divide methods, using both Capped and Uncapped Electoral College totals, to the Popular Vote percentage of their election year

* _Popular_Vote_Diff_NE_and_Divide_ compares the Uncapped Nebraska-Like and Full Divide electoral percentages to the Popular Vote percentages for the Presidential Elections 2008-2020

* _PP_Graphs_and_Results_ is a PowerPoint presentation putting the images into context with the result of the analysis project
