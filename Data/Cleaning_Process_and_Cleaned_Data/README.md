## The **Cleaning_Process_and_Cleaned_Data** sub-folder contains

* _2008_2020_Multiple_Election_Cleaning_File.sql_ which details the SQL code needed to transform the CSV files from "Data\Raw_Data\Isolated_Raw_Data" into a SQL database with cleaned tables for transfer and analysis

* _Cleaning_Process.db_ is the SQL database holding the isolated data in tables and the final cleaned transfer tables (_Transfer_Table_2008_, _Transfer_Table_2012_, _Transfer_Table_2016_, and _Transfer_Table_2020_)
