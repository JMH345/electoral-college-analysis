### The **Analysis_Code** folder contains the main analysis code for comparing the Standard Winner-Take_All, Nebraska-Like, and Full Divide methods within the Capped and Uncapped Electoral Colleges to the national Popular Vote. The four different code files are provided for use depending on user needs when examining or utilizing this code.

* _08_20_DB_construction_code_ is the full computational code to create SQL databases and tables for this analysis project

* _General_analysis_DB_code_ is the full computational code to create SQL databases and tables for any election with data cleaned in the manner demostrated in the **Data** folder

* _Python_code_isolated_ shows the part of the code written in the Python programming language

* _SQL_code_isolated_ shows the part of the code written in the SQL programming language
